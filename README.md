# General information

This repository holds the components that make up my rendered personal website. This is my own personal repository and I am the sole contributor. 

# Data and file information

This repository holds different file types that create the pages, formatting, and structure of my website.
- _quarto.yml organizes the pages and sets a custom website theme and font. It can be found in the main directory
- styles.scss holds the specifications of my custom website theme and font. It can be found in the main directory
- all .qmd files hold the code to each page on my website. Main pages can be found in the main directory. All "project"" pages can be found within the 'projects' folder.
- all rendered .html pages can be found in the 'docs' folder.
- the data for my interactive map can be found in the 'data' folder. 
- all images on my website can be found in the 'images' folder.
